ทดสอบ FPS / ประสิทธิภาพ MuMu Player

อยากรู้ไหมว่าพีซีเครื่องนี้เล่นเกม MuMu Player ได้ลื่นหรือเปล่า? เครื่องมือบนเบราว์เซอร์ฟรีนี้วัดเฟรมเรต (FPS) แบบเรียลไทม์ด้วย requestAnimationFrame และอ่านข้อมูลพื้นฐานของอุปกรณ์ — จำนวนเธรด CPU แรม และ GPU — เพื่อประเมินประสิทธิภาพอย่างรวดเร็ว ทุกอย่างทำงานในเครื่องและไม่มีการอัปโหลดข้อมูลใด ๆ

เคล็ดลับ: เปิดแท็บนี้ไว้ด้านหน้าระหว่างวัด FPS ของเบราว์เซอร์ถูกจำกัดด้วยอัตรารีเฟรชของจอ จึงเป็นแค่ค่าอ้างอิงเชิงเปรียบเทียบ ไม่ใช่ขีดสูงสุดของ GPU

ตัวเลข FPS ของคุณหมายความว่าอะไร

30 FPS คือขั้นต่ำที่พอเล่นได้ การขยับขึ้นเป็น 60 FPS คือก้าวที่รู้สึกถึงความลื่นได้ชัดที่สุด ส่วนเกิน 120 FPS ความต่างจะเล็กน้อย ที่ 60 FPS แต่ละเฟรมมีงบเวลาราว 16.7 มิลลิวินาที สิ่งใดที่ทำให้เฟรมสะดุดจะปรากฏเป็นอาการกระตุก

FPSความรู้สึก
< 30กระตุกชัด เล็งหรือเลื่อนไม่ลื่น
30–45พอเล่นได้ เหมาะกับเกมแคชวลและแบบผลัดตา
60ลื่น เป็นเป้าหมายที่สบายสำหรับเกมส่วนใหญ่
90–120ลื่นมาก ต้องใช้จอ 120 Hz ขึ้นไปและ GPU แรง

MuMu Player จะลื่นบนพีซีของคุณไหม?

MuMu Player ต้องการอย่างน้อย CPU 4 คอร์ที่เปิด VT, แรม 4 GB และ GPU ระดับ GTX 950 อาการกระตุกส่วนใหญ่มาจาก VT ปิดอยู่ จัดสรร CPU หรือแรมให้น้อยเกินไป หรือโหมดเรนเดอร์กราฟิกไม่เหมาะ — คู่มือปรับแต่งประสิทธิภาพและตั้งค่าเฟรมสูงของ MuMu อธิบายวิธีแก้แต่ละข้อทีละขั้น

เช็กเร็วจากสถานการณ์จริง

  • เบราว์เซอร์จำกัดที่ 60 FPS ทั้งที่ GPU แรง? นั่นคืออัตรารีเฟรชจอหรือการจำกัดของเบราว์เซอร์ ไม่ใช่ขีดจำกัดของอีมูเลเตอร์
  • ลื่นบนเบราว์เซอร์แต่กระตุกใน MuMu? ตรวจว่าเปิด VT (เวอร์ชวลไลเซชัน) ใน BIOS แล้วหรือยัง — เป็นสาเหตุที่พบบ่อยที่สุด
  • กระตุกตอนเปิดหลายหน้าต่าง: ลดคอร์และแรมต่อหน้าต่าง หรือเปิดพร้อมกันน้อยลง
  • โหลดกระตุกบนฮาร์ดดิสก์จานหมุนเก่า? ติดตั้ง MuMu Player ลง SSD
  • GPU แสดงเป็น SwiftShader หรือซอฟต์แวร์? แสดงว่าการเร่งด้วยฮาร์ดแวร์ปิดอยู่ เปิดเพื่อให้อ่านค่าแม่นและเกมลื่นขึ้น

คำถามที่พบบ่อย

การทดสอบ FPS นี้ทำงานอย่างไร?
มันใช้ callback requestAnimationFrame ของเบราว์เซอร์เพื่อนับว่าเรนเดอร์ได้กี่เฟรมต่อวินาที ทุกอย่างทำงานในเครื่องบนเบราว์เซอร์ของคุณ ไม่มีการอัปโหลด
ทำไม FPS ของฉันจำกัดที่ 60?
เบราว์เซอร์ซิงค์การวาดกับอัตรารีเฟรชของจอ จอ 60 Hz จะจำกัดเบราว์เซอร์ราว 60 FPS แม้ GPU จะวาดได้มากกว่า
FPS เบราว์เซอร์สูงแปลว่าเกม MuMu จะลื่นแน่ไหม?
เป็นเพียงตัวแทนคร่าว ๆ ของประสิทธิภาพอุปกรณ์ ความลื่นของอีมูเลเตอร์ยังขึ้นกับ VT, CPU และแรมที่จัดสรร และโหมดเรนเดอร์กราฟิก
สเปกขั้นต่ำของ MuMu Player คืออะไร?
ตามข้อกำหนดทางการ: Windows 7 ขึ้นไป, CPU 4 คอร์อย่าง i5-7500 ที่เปิด VT, GPU ระดับ GTX 950 และแรม 4 GB
ทำไม MuMu Player ถึงกระตุก?
พบบ่อยสุดคือ VT ปิด, หน้าต่างได้คอร์หรือแรมน้อยเกินไป หรือโหมดเรนเดอร์ไม่เข้ากับ GPU คู่มือประสิทธิภาพที่ลิงก์ไว้อธิบายวิธีแก้แต่ละข้อ
ได้ 120 FPS ใน MuMu Player ไหม?
บางเกมรองรับโหมดเฟรมสูง คุณยังต้องมีจอ 120 Hz ขึ้นไปและ GPU ที่แรงพอจึงจะเห็นได้จริง
เครื่องมือนี้อัปโหลดข้อมูลของฉันไหม?
ไม่ การวัด FPS และการอ่านอุปกรณ์ทำงานทั้งหมดในเบราว์เซอร์และไม่เคยส่งไปยังเซิร์ฟเวอร์
GPU แสดงเป็นซอฟต์แวร์หรือ SwiftShader หมายความว่าอย่างไร?
เบราว์เซอร์ของคุณไม่ได้ใช้การเร่งด้วยฮาร์ดแวร์ เปิดในการตั้งค่าเบราว์เซอร์เพื่อให้อ่านค่าแม่นและอีมูเลเตอร์ลื่นขึ้น
