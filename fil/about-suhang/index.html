Tungkol sa may-akda: Su Hang

Si Su Hang ang may-akda ng FPS / performance test na ito ng MuMu Player. Ginawa ang tool para matulungan ang mga manlalaro na mabilis matantya kung kaya ng PC nila na patakbuhin nang maayos ang mga Android game sa emulator.

Lahat ng datos ng tool ay tinipon mula sa opisyal na system requirements ng MuMu Player at pampublikong sanggunian ng frame rate, at para sa sanggunian lamang.
