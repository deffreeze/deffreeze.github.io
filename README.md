# deffreeze.github.io

MuMu Player FPS & Performance Test — a free, client-side browser tool that measures your live frame rate (FPS) with requestAnimationFrame and reads your CPU, RAM and GPU to estimate whether Android games run smoothly in the MuMu Player emulator.

Live: https://deffreeze.github.io/  ·  12 languages (zh-Hans/zh-Hant/en/ja/id/th/vi/ms/fil/km/lo/my).

Data compiled from MuMu Player official requirements and public frame-rate references. MIT License.
